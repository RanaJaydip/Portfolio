# Portfolio (Next.js + React)

Modern, responsive developer portfolio for **mobile + desktop**.

## Run locally

1) Install deps

```bash
npm install
```

2) Start dev server

```bash
npm run dev
```

Open `http://localhost:3000`.

## Customize

- Edit your info in `src/data/site.ts`
- Edit projects in `src/data/projects.ts`
- Edit skills in `src/data/skills.ts`
